# Source Files

## Files

- `Document.java`: Common document interface.
- `WordDocument.java`, `PdfDocument.java`, `ExcelDocument.java`: Concrete document classes.
- `DocumentFactory.java`: Abstract creator class.
- `WordDocumentFactory.java`, `PdfDocumentFactory.java`, `ExcelDocumentFactory.java`: Concrete factories.
- `FactoryMethodTest.java`: Main class that demonstrates factory-created documents.
